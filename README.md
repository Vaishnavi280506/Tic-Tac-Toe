# Tic-Tac-Toe

A classic Tic-Tac-Toe game built with HTML, CSS, and JavaScript — no frameworks, just clean vanilla code.

## What I focused on

- DOM manipulation to handle game state and player turns
- Win detection logic covering all rows, columns, and diagonals
- Responsive UI that works on both desktop and mobile
- Clean, readable code structure across separate HTML, CSS, and JS files

## How to run

1. Clone the repo or download the files
2. Open `Tic Tac Toe.html` in your browser
3. That's it — no installs needed
